1.T아이디 Logo 사용 가이드

로고 타입 적용 시 그에 따른 규정과 원칙을 반드시 준수하고,
왜곡, 변경, 오남용 등 어떠한 이미지 손상도 발생시키지 않도록 합니다.

Incorrect Usage

현상을 변경하거나 가공하면 심볼 마크의 느낌이 훼손됩니다. 본 가이드라인을 참고하여 올바르게 사용합니다.

Don't

Don't

Don't

Symbol Logo

가이드를 벗어난 디자인은 T아이디의 아이덴티티를 약화시키므로 해당 가이드라인을 참고하여 올바르게 사용합니다.

Don't

T아이디 심볼로고가 잘못 적용된 경우 / T아이디 심볼 로고 옆 이름이 표시 된 경우

Do

T아이디 신규 심볼로고 적용
T아이디 심볼 로고 옆 및 아이디 표시 영역에는 아이디값만 표시 (이름 표시 불가능)
- T아이디 3종 : 휴대폰 번호 아이디, 이메일 주소 아이디, Tworld 아이디

2.T아이디 로그인 버튼 사용 가이드

T아이디 로그인 버튼은 T아이디 고유의 이미지를 해치지 않는 범위 내에서 유연하게 디자인을 변경할 수 있습니다.

기본 버튼 형태를 유지한 상태로 BG컬러에 따라 라인으로 처리될 수 있습니다.

Usage

가이드를 벗어난 디자인은 T아이디의 아이덴티티를 약화시키므로 해당 가이드라인을 참고하여 올바르게 사용합니다.

Don't

다른 서비스 로그인 버튼 크기와 스타일이 상이한 경우

Do

시각적으로 유사한 효과 나타나도록 다른 서비스 로그인 버튼의 크기와 스타일 비슷하게 적용

Incorrect Usage

Don't

신규 T아이디 로고를 적용하지 않는 경우

Don't

T아이디 버튼 컬러코드를 준수하지 않는 경우

Don't

신규 T아이디 로고를 적용하지 않는 경우

3.T아이디 회원정보

T아이디 회원정보 관리 화면을 애플리케이션에 적용하여 활용하실 경우
“T아이디 > 회원정보 관리" 화면의 일관된 메뉴 / 버튼명 통일을 위해 명시된 규정에 따라 올바르게 사용합니다.
가이드를 벗어난 다른 표기는 금합니다.

Don't

회원정보 관리 / T아이디 + 아이디값 형태로 혼용한 경우
회원정보 조회 / 회원정보 수정 / 회원정보 변경 / 내 정보 수정 등 다른 레이블을 표기한 경우

Do

T아이디 > 회원정보 관리 화면으로 연결된 메뉴/버튼 레이블은 [T아이디 회원정보]로 표시
- 공간 제약이 있어 짧게 표시해야 하는 경우 [회원정보]로 표시 가능
- 메뉴/버튼 내에 '아이디값' 표시 가능

4.애플리케이션 앱 아이콘 적용 가이드

애플리케이션 앱 아이콘은 '정보 동의 화면'과 'T아이디 포털의 T아이디로 이용 중인 서비스와 이용 가능한 서비스'에서
애플리케이션의 앱 아이콘이 노출됩니다. T아이디 포털 https://www.t-id.co.kr
